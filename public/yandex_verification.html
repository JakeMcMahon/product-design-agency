Верификация Yandex Webmaster

Эта страница используется для верификации сайта в Yandex Webmaster Tools.

Инструкция по настройке:

  1. Добавьте сайт ai.productquant.ru в Yandex Webmaster
  2. Выберите метод подтверждения "HTML-файл"
  3. Скачайте файл подтверждения и замените этот файл
  4. Или добавьте мета-тег verification в head секцию index.html
